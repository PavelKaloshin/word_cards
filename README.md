# Serbian Cards

Локальное web-приложение для изучения сербских слов с интервальным повтором.

## Что умеет

- Добавление слов **списком** или **скриншотом** (OCR через OpenAI vision).
- Для каждого слова автоматически: перевод и пример в контексте (gpt-4o-mini), картинка из Wikipedia/Commons, аудио через edge-tts.
- Два режима сессии:
  - **Новые** — Leitner-style очередь, идёт пока все новые слова не запомнятся.
  - **Повтор** — взвешенный случайный выбор слов с приоритетом ошибочных и просроченных.
- Три кнопки оценки: **Again / Hard / Good**.
- Прямое (картинка → слово) и обратное (слово → картинка) направление.
- Кириллица / латиница / оба алфавита одновременно (тоггл `c`).
- Опциональный режим ввода с клавиатуры с автоматической оценкой по Levenshtein.
- HUD сессии в углу, история сессий, настройки скоринга.
- Все действия по клавиатуре. Подсказки хоткеев на экране, оверлей по `?`.

## Установка

```bash
cd ~/projects/serbian-cards
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Установи переменную окружения:

```bash
export OPENAI_API_KEY="sk-..."
```

(или положи в `~/.zshrc`).

## Запуск

```bash
python run.py
```

Откроется `http://127.0.0.1:8765`. Браузер запустится автоматически.

## Хоткеи

### Главный экран
- `g` — новые слова
- `r` — повтор
- `a` — добавить слова
- `,` — настройки
- `h` — история сессий
- `?` — справка

### Сессия
- `Space` или клик по карточке — перевернуть
- `1` / `2` / `3` (или `j` / `k` / `l`) — Again / Hard / Good
- `s` — озвучить
- `t` — показать перевод
- `e` — показать пример
- `Shift+E` — сгенерировать новый пример
- `c` — переключить алфавит (кир / лат / оба)
- `i` — режим ввода с клавиатуры
- `Esc` — выйти из сессии (с записью результатов)

### Добавление слов
- `Cmd+V` / `Ctrl+V` — вставить скриншот
- `Esc` — назад

## Персистентность

Каждое изменение (`add_word`, `mark_answer`, `update_setting`) атомарно дампится в JSON через `tempfile + os.replace`. Никакой ручной "save" кнопки — данные всегда на диске.

Картинки и аудио лежат в `data/media/`, в JSON хранятся только пути. При удалении слова файлы тоже удаляются.

## Скоринг

См. [SCORING.md](SCORING.md). Все константы вынесены в `config.json` и доступны через UI настроек.

## Стоимость

- Текст (перевод + пример): 1 вызов gpt-4o-mini на слово — копейки.
- Скриншот при добавлении: 1 vision-вызов gpt-4o-mini, дёшево.
- Картинки: бесплатно (Wikipedia/Commons).
- Аудио: бесплатно (edge-tts).

Если для слова не нашлось картинки в Wikipedia — карточка просто покажется без неё; кнопка "↻" (refind) попробует ещё раз. Можно поменять язык поиска через `image_search_lang` (например, на `"ru"` если перевод на русском).
