# UX & Hotkeys

Принцип: **всё через клавиатуру**. Хоткеи всегда видны на экране (подсветка букв на кнопках, либо пометка `[Space]` рядом).

## Карта хоткеев

### Везде
- `?` — показать оверлей со всеми хоткеями текущего экрана.
- `Esc` — выйти на главный экран.
- `g` — перейти в режим "новые".
- `r` — перейти в режим "повтор".
- `a` — добавить слова (форма + drop zone для скриншота).
- `,` — настройки.

### Карточка во время сессии (front side)
- `Space` — перевернуть карточку (показать обратную сторону).
- `s` — озвучить (speak).
- `t` — показать перевод (toggle).
- `e` — показать пример в контексте (toggle).
- `Shift+E` — сгенерировать новый пример.
- `c` — переключить алфавит (кириллица / латиница / оба).
- `i` — переключиться в режим ввода с клавиатуры (если включён в конфиге).

### Карточка после переворота (back side)
- `1` — **Again** (не угадал)
- `2` — **Hard** (сложно)
- `3` — **Good** (угадал)

(Дублирующие: `j` / `k` / `l` для тех, кто на ноуте без правильного нумпада. Конфигурируется.)

### Режим ввода (когда `i` нажата или `typing_mode_enabled = true`)
- Вводишь слово, `Enter` — отправить ответ.
- `Tab` — сдаться, посмотреть правильный ответ (засчитывается как **again**).
- После показа результата автогрейдинга — `1`/`2`/`3` могут переопределить, если хочешь поправить (например, посчитал хард как гуд).

### Форма добавления слов
- `Cmd+V` (Mac) / `Ctrl+V` — вставить скриншот из буфера.
- `Cmd+Enter` — submit формы.
- `Tab` — переход между полями.

## Подсветка хоткеев на экране

Каждая кликабельная кнопка показывает рядом или поверх свою клавишу:

```
┌─────────────────┐
│  Again      [1] │
└─────────────────┘
```

Стиль: маленький badge с буквой/цифрой в углу кнопки, серым фоном, моноширинным шрифтом.

Оверлей `?` — модалка со всеми хоткеями текущего экрана списком, закрывается по `Esc` или повторному `?`.

## Поведение сессии

### Старт сессии
Главный экран показывает:
- Кнопка "Новые слова" с числом доступных новых слов. Disabled, если новых нет.
- Кнопка "Повтор" с числом слов "пора повторить" (где `due_factor ≥ 1`).
- "Все настройки" (хоткей `,`).

### Внутри сессии (HUD в углу)
В левом верхнем углу всегда видна статистика **этой сессии**:
```
✓ 12   ✗ 3   acc 80%   17/100
```
- ✓ — good + hard
- ✗ — again
- acc — accuracy
- N/M — текущая позиция / размер сессии (для "новых" — N/total_new)

### Конец сессии
Экран с итогами:
- Сколько слов прошло / правильно / неверно.
- Сколько стало mastered за эту сессию.
- Какие слова дались хуже всего (топ-5 по ошибкам в сессии) — для самоанализа.
- Кнопки: "ещё одна сессия", "на главный".

## Экран добавления слов

Два таб-входа в одном UI:

1. **Список** — textarea, по строке на слово. Можно `сербское_слово | перевод` или просто слово (перевод/значение сгенерим LLM-ом).
2. **Скриншот** — drop zone / paste. Через OpenAI vision: распознать список слов с переводами.

После парсинга → preview таблицей → "сохранить все" / убрать галочки с тех, что не надо.

При сохранении для каждого слова сразу:
- Генерируем перевод (если не указан).
- Генерируем пример в контексте.
- Генерируем/находим картинку.
- Сохраняем всё в `media/` + JSON.

Прогресс-бар внизу (генерация может занять секунд 5-10 на слово).
